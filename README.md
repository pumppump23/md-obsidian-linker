# MD Obsidian Linker

PWAアプリ。指定文字列をObsidianの`[[...]]`形式に変換し、.mdファイルの読み書き・コピーができます。

## GitHub Pages
https://pumppump23.github.io/md-obsidian-linker/

Safari / Chrome で開くとオフラインでも動作します。iPhoneの場合は「ホーム画面に追加」でネイティブアプリ風に使用できます。

## 使い方
1. 「.mdファイルを読み込む」でファイルを選択
2. 「リンク化する文字列」にキーワードを入力
3. 「変換 & プレビュー」
4. 「コピー」または「.mdダウンロード」
